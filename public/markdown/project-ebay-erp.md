# 电商 ERP

eBay 跨境电商多店铺管理后台——商品同步、销售分析、订单自动化，一个界面搞定。

🔗 **在线体验**: [erp.minolab.icu](https://erp.minolab.icu)

---

## 亮点

- 🏪 **多店铺管理** - 统一管理多个 eBay 账号
- 📦 **商品同步** - 批量上架与库存同步
- 📊 **数据分析** - 销售报表与利润统计
- 🔄 **订单处理** - 自动化发货与物流跟踪

## 技术栈

- React + TypeScript
- Ant Design 组件库
- RESTful API
- 响应式 Dashboard 布局
