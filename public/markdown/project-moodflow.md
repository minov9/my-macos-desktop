# MoodFlow

治愈系情绪日记 App——珊瑚暖色调设计，记录每日心情、可视化趋势、养成健康情绪习惯。

📱 **平台**: iOS / Android　　🔗 **开源**: [GitHub](https://github.com/minov9/MoodFlow)

---

## 亮点

- 📊 **情绪趋势** - 折线图 + 环形图，一眼看清心情变化
- 📅 **日历标记** - 渐变色日历，每日情绪一目了然
- 🎨 **Apple Design** - 珊瑚暖色调 (#FF7B6B)，圆角卡片 + 柔和阴影
- 🌐 **中英双语** - 完整 i18n 国际化支持
- 🔒 **隐私优先** - 数据仅存本地，无云端同步

## 技术栈

- Flutter 3.x — 跨平台框架
- Riverpod — 状态管理
- Hive — 轻量本地存储
- fl_chart — 数据可视化
- go_router — 声明式路由
