# Mino's Desktop

仿 macOS 风格的个人主页——没错，就是你现在正在浏览的这个！

🔗 **在线访问**: [minolab.icu](https://minolab.icu)

---

## 亮点

- 🖥️ **真实桌面体验** - 完整还原 macOS 的窗口、Dock、菜单栏
- 🪟 **窗口管理系统** - 支持拖拽、缩放、最小化
- 🎨 **深色/浅色主题** - 跟随系统偏好自动切换
- 📝 **多应用集成** - Bear、Terminal、Safari 等经典应用模拟

## 技术栈

- React 18 + TypeScript
- Zustand 状态管理
- UnoCSS 原子化样式
- Framer Motion 动效
- Vite 构建工具
