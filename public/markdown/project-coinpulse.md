# CoinPulse

iOS 原生加密货币追踪 App——实时行情、走势图表、投资组合管理，零第三方依赖，纯 SwiftUI 打造。

🍎 **平台**: iOS 16.0+　　🔗 **开源**: [GitHub](https://github.com/minov9/CoinPulse)

---

## 亮点

- 📈 **实时行情** - CoinGecko API 驱动，7 天走势图 + 市场统计
- 💼 **组合管理** - 添加/编辑持仓，实时计算盈亏
- 🌙 **深浅色模式** - 完整适配 Dark Mode
- 🚫 **零依赖** - 100% 原生 Swift，无第三方库

## 技术栈

- SwiftUI — 声明式 UI 框架
- Combine — 响应式数据流
- CoreData — 本地持久化
- MVVM — 架构模式
