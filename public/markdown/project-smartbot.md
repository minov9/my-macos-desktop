# SmartBot

微信小程序里的 AI 客服——扫码即聊，Apple 风格 UI，支持 DeepSeek / OpenAI 等多模型实时对话。

📱 **平台**: 微信小程序

---

## 亮点

- 🤖 **AI 智能对话** - 基于大语言模型的智能问答
- ⚡ **极速响应** - 流式输出 + 打字机效果，毫秒级反馈
- 🔧 **多服务商支持** - DeepSeek / OpenAI / 自定义 API
- 🎨 **Apple 风格 UI** - 简洁优雅的 iMessage 风格界面
- 📦 **零后端部署** - 小程序直连 AI API，无需服务器

## 技术栈

- 微信原生小程序 — 高性能轻量级框架
- DeepSeek / OpenAI API — 多模型支持
- 流式输出 — 实时打字机效果
- Apple Design — iOS 风格交互设计
