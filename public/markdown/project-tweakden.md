# TweakDen

小红书笔记体检工具——发布前一键检测违禁词、限流风险、标题评分，帮助创作者提升笔记质量。

🔗 **在线访问**: [tweakden.com](https://tweakden.com)

---

## 亮点

- 🔍 **违禁词检测** - 实时扫描笔记内容，标记敏感词与风险表述
- 📊 **AI 标题评分** - 基于 DeepSeek 大模型分析标题吸引力并给出优化建议
- ⚠️ **限流风险分析** - 评估笔记被限流的潜在风险
- 💳 **完整商业闭环** - 从产品定义到支付变现的全流程独立完成

## 技术栈

- Next.js (App Router) — 全栈框架
- PostgreSQL — 数据持久化
- DeepSeek API — AI 分析引擎
- NextAuth — 用户认证
- ZPay — 支付集成
- Vercel — 部署平台
