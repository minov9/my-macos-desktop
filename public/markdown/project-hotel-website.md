# 澄境雅舍

五星级海滨度假酒店官网，诗意江南与现代奢华的融合。全屏大图、视差滚动、中式美学。

🔗 **在线访问**: [hotel.ruinarc.com](https://hotel.ruinarc.com)

---

## 亮点

- 🏨 **沉浸式展示** - 全屏大图与视差滚动效果
- 🍽️ **多业态呈现** - 客房、餐饮、SPA、会议一站式展示
- 📅 **在线预订系统** - 日期选择与房型筛选
- 🌊 **中式诗意设计** - 融合江南水乡美学

## 技术栈

- React + TypeScript
- CSS Grid & Flexbox
- Intersection Observer 滚动动效
- 响应式设计
