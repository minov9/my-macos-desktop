# Mino's Desktop

A macOS-inspired personal portfolio website with an immersive desktop experience. Drag, resize, and interact with windows just like a real operating system — all in your browser.

**Live Demo**: [minolab.icu](https://minolab.icu)

## Features

- 🖥️ **macOS Window System** — Draggable, resizable windows with minimize/maximize support
- 🌗 **Dark & Light Mode** — Full theme switching, matching macOS aesthetics
- 🎵 **Music Player** — Built-in lofi music player with visualizer
- 📝 **Bear Notes** — Project showcase in a note-taking app style
- 🔍 **Spotlight Search** — Quick navigation across all apps
- 📱 **Launchpad** — App grid with smooth animations

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | React 18 + TypeScript |
| State | Zustand |
| Styling | UnoCSS |
| Animations | Framer Motion |
| Build | Vite |
| Deploy | Vercel |
