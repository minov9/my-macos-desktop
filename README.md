# Mino's Desktop

仿 macOS 风格的个人作品集网站，沉浸式桌面体验。

**Live Demo**: https://minolab.icu

---

## 技术栈

- React + TypeScript
- Zustand 状态管理
- UnoCSS 原子化样式
- Vite 构建工具

---

## 开发

```bash
# 安装依赖
npm install --legacy-peer-deps

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
